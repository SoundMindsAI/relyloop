## Known debt / fragility

Any stack up >24h silently loses all LLM-dependent capability (UBI hybrid + LLM judgment generation, digest narrative, chat tool dispatch) ‚Äî endpoints return `503 LLM_PROVIDER_INCAPABLE "cache miss"` until the API process is restarted. Confirmed live 2026-06-02 (operator stack at 34h uptime had zero `openai:capabilities:*` keys). **Operational workaround: `docker compose restart api`** repopulates the cache.
